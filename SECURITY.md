# Security Policy

このプロジェクトは教育・創作向けのエミュレーターです。生成されたコードや
Playgroundは信頼できない入力を扱う可能性があるため、次を前提にしてください。

- Playgroundサーバはローカル開発用です。公開インターネットへ直接さらさないでください。
- C風コンパイラは独自言語を解析し、Pythonコードとして実行しません。
- 実行ステップ上限により無限ループを止めます。
- 外部監査や外部サービスへワークスペース内容を送る場合は、送信範囲を明示してください。

脆弱性や危険な挙動を見つけた場合は、公開Issueに詳細な攻撃手順を書かないでください。
リポジトリの **Security** タブにある **Report a vulnerability** から、GitHub の
Private Vulnerability Reporting を使って非公開で連絡してください。再現手順、影響範囲、
修正案があれば一緒に記載してください。
