# Technical Repository Scope

This repository distributes only the technical E-base Computer implementation:

- emulator and C-like compiler source code;
- executable instruction and state specifications;
- deterministic behavior models derived from the released runtime, including
  thermal, quantization, observation, refresh, and scoring rules;
- Web Playground, examples, tests, and challenge tooling; and
- technical contributor and release documentation.

These materials are licensed under Apache-2.0. See [LICENSE](LICENSE),
[NOTICE](NOTICE), and [TRADEMARKS.md](TRADEMARKS.md).

Creative worldbuilding, narrative history, in-universe institutions, and
nonessential setting material are deliberately not included in this technical
distribution. They are not licensed by this repository.

The distinction is functional: documentation required to implement, test,
predict, or optimize the emulator is technical material. See
[docs/behavior_model.md](docs/behavior_model.md). Narrative meaning beyond that
runtime contract remains outside this repository.
